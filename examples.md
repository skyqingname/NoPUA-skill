# 标注示例集

> 2 个完整示例，展示技能的三阶段输出流程。每个示例包含输入、Phase 1/2/3 完整输出及标注说明。

---

## 示例 1：加班绑架 + 公开施压

### 输入
> 「你不加班？大家都在加班，你这样让团队怎么看你？」

**场景**：周五下午6点，领导当着3个同事的面说

---

### Phase 1 输出（话术拆解）

```
── 话术拆解 ──
🔍 识别到的操控模式：加班文化绑架（Overtime Culture Coercion）
• 具体手法：用"大家都在加班"制造群体压力，将个人的正常下班行为框定为对团队的背叛。
• 操控机制：不直接命令加班，而是通过暗示"所有人都在做"来制造从众压力。不加班者不会被明确处罚，但会被贴上"不合群"的隐性标签，这种软强制比硬性命令更难反抗——因为"没有人逼你"。
• 关键破绽：工作成果的衡量标准应该是产出质量，不是在工位上的时长。"大家都在加班"是事实陈述还是施压工具，取决于说话者是否允许你选择不加班而不承受后果。

🔍 识别到的操控模式：公开羞辱/杀鸡儆猴（Public Shaming）
• 具体手法：选择当着3个同事的面质问，而非私下沟通，将个人的工作安排变成公开审判。
• 操控机制：公开场合的质问同时攻击了尊严和社会形象。被质问者在同事面前很难从容回应，旁观的同事也会因此更不敢准时下班。一次公开施压可以换来整个小组长期的"自愿加班"。
• 关键破绽：如果目的是讨论工作安排，私下沟通效率更高。选择当众说，说明目的不是解决问题，而是通过制造社交压力来迫使服从。
```

---

### Phase 2 输出（情绪验证）

```
── 你的感受 ──
你感到羞耻和被困住，这是正常的反应。
对方当着同事的面质问你，这直接攻击了你的社会形象——你的不适不是"玻璃心"，而是任何人在公开被指责时都会有的正常应激反应。同时，"大家都在加班"这句话制造了一种道德压力，让你觉得准时下班是一种亏欠，这种内疚感是被设计出来的，不是你真的做错了什么。
这不是你的问题。准时下班是劳动合同赋予你的权利，不需要向任何人证明合理性。
```

---

### Phase 3 输出（回应参考）

```
── 回应参考 ──
如果你想回应，可以参考：
「我今天的工作已经完成了。如果有额外的任务需要协调，我们可以明天单独聊一下排期。」

原理：这个回应直接回应了"加班文化绑架"的逻辑破绽——将讨论从"你为什么不加班"拉回到"工作是否完成"这个真正的衡量标准。同时，"单独聊"隐含地拒绝了公开审判的场域，把对话拉回私下沟通的正常轨道。

⚠️ 这只是参考。是否回应、怎么回应，取决于你对自己处境的判断。
```

> 注意：此场景中领导当众施压，且权力关系不对等。如果你判断正面回应风险较高，选择不回应也完全合理——沉默不等于认同。

---

### 标注说明

**为什么识别为这两个模式：**
- "大家都在加班"是加班文化绑架（5.2）的典型标记——不直接命令，而是用群体行为制造隐性强制。关键词"大家都在"触发从众压力机制。
- "当着3个同事的面"是公开羞辱（3.3）的场景标记——选择公开场合而非私下沟通，目的是通过社交压力放大服从效果。即使语气不算激烈，公开质问本身就构成施压。

**两个模式的叠加效应：**
加班绑架提供了"内容层"的压力（你应该加班），公开羞辱提供了"场域层"的压力（你现在不能反驳）。两者叠加，使得被施压者既在道德上觉得"我确实不该走"，又在社交上觉得"我现在没法解释"。这种组合比单独使用任何一种都更有效。

**这个示例的教学价值：**
这是一句非常日常的话，很多人听到后只会觉得"不舒服"但说不清为什么。拆解后可以看到，短短一句话里同时运作了两种不同维度的操控机制。识别的关键不只是话术内容，还包括说话的场景（公开 vs 私下）。

## 示例 2：绩效面谈中的复合操控

### 输入
> 「你这个方案不行，但我也不怪你，毕竟你经验还不够。我当年像你这么大的时候，早就能独立带项目了。」

**场景**：一对一绩效面谈

---

### Phase 1 输出（话术拆解）

```
── 话术拆解 ──
🔍 识别到的操控模式：比较贬低（Comparative Devaluation）
• 具体手法："我当年像你这么大的时候，早就能独立带项目了"——用自己的过去作为标杆，暗示对方远远不如。
• 操控机制：比较制造的不是动力，而是羞耻感。受害者的注意力从"方案哪里需要改进"转移到"我是不是不如领导当年"，陷入自我否定。而且，领导对自己过去的描述无法验证，比较的基准本身就不公平。
• 关键破绽：每个人的背景、资源、时代环境不同，跨时代的横向比较没有意义。而且，如果领导当年真的那么强，为什么不能具体指导方案的改进方向，而只是用比较来施压？

🔍 识别到的操控模式：「为你好」话术（Paternalistic Manipulation）
• 具体手法："我也不怪你"——表面上是宽容，实际上预设了"你有错、我有资格怪你"的权力位置。"不怪你"的潜台词是"我本可以怪你"，这是一种居高临下的施恩姿态。
• 操控机制：通过"不怪你"的表态，将批评包装成宽容，使对方难以反驳——如果你反驳，就变成了"人家都不怪你了你还不知足"。这种话术借用了长辈式的权威感，让控制行为看起来像关怀。
• 关键破绽："不怪你"预设了对方有过错且自己有裁判权，但方案是否"不行"需要具体论证，不是一句话就能定性的。真正的反馈会说"方案的X部分需要改进"，而不是先下结论再"宽恕"。

🔍 识别到的操控模式：贴标签（Labeling）
• 具体手法："毕竟你经验还不够"——将方案的问题归因为个人能力/经验的全面不足，而非方案本身的具体缺陷。
• 操控机制：一旦接受了"经验不够"这个标签，所有后续的分歧都可以被归结为"你还嫩"。这个标签会让受害者在未来的讨论中自动降低自己的发言权重，因为"我经验不够，也许领导说得对"。
• 关键破绽：从"这个方案有问题"到"你经验不够"是一个逻辑跳跃。方案不好可能是需求不清晰、时间不够、资源不足等多种原因，直接归因为"经验不够"是以偏概全。
```

---

### Phase 2 输出（情绪验证）

```
── 你的感受 ──
你感到自我怀疑和挫败，这是正常的反应。
对方用"经验不够"给你贴了标签，又用自己的过去作为不可验证的标杆来比较——这两招叠加，会让你觉得"也许我真的不行"。同时，"我也不怪你"这种居高临下的宽容，让你连反驳的空间都没有——因为对方已经站在了"原谅你"的位置上。
这不是你的问题。方案需要改进是具体的工作议题，但对方没有给出任何具体的改进方向，而是把问题归结为你这个人"经验不够"。这说明问题出在反馈方式上，不是你的能力上。
```

---

### Phase 3 输出（回应参考）

```
── 回应参考 ──
如果你想回应，可以参考：
「方案具体哪些地方需要调整？我想听一下具体的改进方向。」

原理：这个回应直接回应了"贴标签"的逻辑破绽——将对话从"你这个人行不行"拉回到"方案哪里需要改"。它迫使对方给出具体反馈，而不是停留在笼统的人格评价上。同时，语气平和专业，不构成对抗，在一对一绩效面谈的场景中风险可控。

⚠️ 这只是参考。是否回应、怎么回应，取决于你对自己处境的判断。
```

---

### 标注说明

**为什么识别为这三个模式：**
- "我当年像你这么大的时候，早就能独立带项目了"是比较贬低（4.2）的典型标记——用他人（此处是过去的自己）作为标杆来制造"你不如人"的羞耻感。
- "我也不怪你"是「为你好」话术（5.1）的变体——表面宽容，实际预设了裁判权和过错归属，借用长辈式权威压制异议。
- "毕竟你经验还不够"是贴标签（4.1）的典型标记——将具体问题上升为对人的全面否定性评价。

**三个模式的组合为什么更难识别：**
这段话的危险之处在于它"听起来不像坏话"。"不怪你"听起来像宽容，"经验不够"听起来像客观评价，"我当年"听起来像分享经验。三种话术各自都有"合理"的外衣，叠加在一起形成了一个完整的操控链条：先定性（方案不行）→ 再施恩（不怪你）→ 再贴标签（经验不够）→ 最后用比较锁死（我当年比你强）。受害者很难在短时间内拆解这个链条，往往只会留下一个模糊的感受："领导说得好像也有道理，但我就是觉得不舒服。"

**与示例 1 的对比：**
示例 1 的操控相对"粗暴"——公开施压、直接绑架，容易被识别。示例 2 则更隐蔽，每一句都包裹着"善意"的外壳，需要逐层拆解才能看到操控机制。这也是为什么绩效面谈是职场 PUA 的高发场景——一对一的封闭环境加上权力不对等，使得这类复合话术特别有效。
